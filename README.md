# Client Inquiry Automation (n8n)

This project is an automated workflow built with n8n to handle client inquiries.

## Features
- Captures inquiries via Google Forms
- Stores data in Google Sheets
- Sends real-time Telegram notifications
- Sends automatic email responses

## Workflow
Google Sheets Trigger → Data Processing → Telegram → Gmail

## Tools Used
- n8n
- Google Forms
- Google Sheets
- Telegram Bot API
- Gmail API

## Use Case
- Lead generation
- Client inquiry handling
- Business automation

## Setup
1. Create Google Form
2. Connect Google Sheets
3. Configure n8n nodes
4. Add Telegram bot
5. Connect Gmail
6. Import workflow.json
